# レビュー記入ワークシート（受講者用）

`reveal` を打つ前に、各自このシートを埋めます。1演習につき1枚です。グループ研修では、議論の前に全員が先に書きます。

印刷して使う場合は、以下をコピーして手書きでもかまいません。

## 評価軸の振れ

このコードが8つの評価軸のどれを過大評価（↑）、過小評価（↓）、評価不能にしているかに印を付けます。混入パターンは特定の軸に偏るので、賭けたパターンの偏り（カタログの「評価軸の偏り」）と一致するかを確かめます。

| 評価軸 | ↑ 過大 | ↓ 過小 | 評価不能 |
|---|:---:|:---:|:---:|
| 目的適合性 | ☐ | ☐ | ☐ |
| 制約適合性 | ☐ | ☐ | ☐ |
| 整合性 | ☐ | ☐ | ☐ |
| 実現可能性（技術 / 組織） | ☐ | ☐ | ☐ |
| 時間効果（短期 / 長期） | ☐ | ☐ | ☐ |
| 品質影響（保守性 / 運用性 / 変更容易性 / 進化性） | ☐ | ☐ | ☐ |
| リスク・不確実性 | ☐ | ☐ | ☐ |
| 合意可能性 | ☐ | ☐ | ☐ |
